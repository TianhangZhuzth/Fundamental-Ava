# Security

## Audit

AlloyFund contracts were audited prior to mainnet deployment.
Report available on request. No critical, high, or medium findings at launch.

## Bug bounty

Responsible disclosure: open a private GitHub security advisory.

## Known limitations

- The keeper is permissionless but not trustless.
- LP positions are permanently locked — no withdrawal function by design.
- Backing stock oracle uses Robinhood-issued tokenized stock prices (centralized at issuance layer).
